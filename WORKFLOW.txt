AUTONOMOUS OPTIMIZATION LOOP (Every 5 min):

1. BUILD CONTEXT
   ├─ Current temperature
   ├─ Humidity level
   ├─ Occupancy status (motion sensors)
   ├─ Current energy consumption
   ├─ Current time (hour)
   ├─ Weather conditions
   ├─ Device states
   └─ Energy forecast (24h)

2. EVALUATE RULES (Priority Order)
   ├─ Peak Hour Rules (5 PM - 9 PM)
   │  └─ If energy > MIN_THRESHOLD → Reduce by 20%
   │
   ├─ Occupancy Rules
   │  ├─ If occupied → Comfort mode
   │  └─ If unoccupied → Eco mode
   │
   ├─ Weather Rules
   │  ├─ Rain/Snow → Ensure hot water
   │  ├─ Sunny → Use natural light
   │  └─ Cold/Hot → Adjust temperature
   │
   ├─ Comfort Rules
   │  ├─ Temp out of range → Adjust HVAC
   │  └─ Humidity out of range → Activate humidifier
   │
   └─ Off-Peak Rules (11 PM - 7 AM)
      └─ If off-peak → Allow charging

3. GENERATE DECISIONS
   └─ Dict: {device: action}

4. EXECUTE DECISIONS
   └─ Publish to MQTT for each device

5. LOG & TRACK
   └─ Store decision in history
